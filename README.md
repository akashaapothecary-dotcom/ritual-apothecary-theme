# Ritual Apothecary Theme

A custom Shopify Online Store 2.0 theme for handcrafted herbal offerings, ritual boxes, and ceremonial objects for those moving through thresholds of transformation, grief, rebirth, and spiritual becoming.

## Features

- **Dark, Luxurious Aesthetic**: Gold accents, sage greens, deep roses, blackened plum, and cream tones
- **Modular Sections**: Fully customizable homepage sections including hero, portals, featured products, threshold finder, founder statement, and newsletter
- **Ritual-Focused Design**: Product pages with accordion details (ingredients, ritual use, care notes)
- **Mobile Responsive**: Optimized for all devices
- **Accessibility**: WCAG compliant with proper semantic HTML, ARIA labels, and keyboard navigation
- **Customizable Typography**: Elegant serif fonts (Playfair Display, Crimson Text, EB Garamond)

## Installation

1. Connect this repository to your Shopify store
2. Configure colors, fonts, and content via the theme customizer
3. Add sections to your homepage using the JSON editor or drag-and-drop builder

## Sections Included

- **Ritual Hero**: Full-screen header with background image, overlay, and CTA buttons
- **Collage Category Portals**: Grid of category cards with hover effects
- **Featured Ritual Products**: Showcase collection products
- **Threshold Finder**: Six threshold categories with descriptions
- **Ritual Philosophy**: Brand philosophy statement
- **Ritual Objects**: Secondary messaging section
- **Private Threshold Support**: Highlighted call-to-action
- **Founder Artist Statement**: Founder bio with image
- **Apothecary Journal**: Blog/editorial section
- **Newsletter Threshold**: Email signup

All theme colors, spacing, fonts, and transitions can be modified via the CSS custom properties in `assets/ritual-apothecary.css`.
